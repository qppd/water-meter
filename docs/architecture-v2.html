💧 Water Meter — System Architecture

Hardware Stack

  • • 5× YF-S201 Flow Sensors (1 inlet + 4 fixtures)
  • • 4-5× Solenoid Valves (NC · 12V)
  • • OLED 128×64 · SD Card · Buzzer
  • • 4-Ch Relay Module · Check Valves

ESP32 Firmware

  • • Local leak rules (non-ML fallback)
  • • SD Card backup during WiFi outages

ML & Backend

  • • Pyrebase4 stream listener (real-time)
  • • Flask Dashboard + Chart.js visualizations
  • • Daily retrain pipeline · Telegram alerts
